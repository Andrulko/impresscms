XOOPS Core
==========

SmartX is currently based on XOOPS version 2.0.15 (http://devteam.xoops.org/releases/xoops-2.0.15.html)


SmartObject Framework
=====================

The SmartX package features the SmartObject Framework version 1.0.

XOOPS Multilanguages
====================

The SmartX package features the integration of XOOPS Multilanguages 2.0.5 to manage multilingual sites.
