The MESSENGER modules originally started as a fork from mpmanager, from version 2.63. Because of this, many files in the code base comes from Venom (Original_Author) and were originally copyrighted as such.

Every file that was originally created by Venom (Original_Author) but then edited by The ImpressCMS Project has seen its header comment block edited, to replace the original author comment block by a new smaller one. Every single file created by the author but edited by ImpressCMS is and will always display in its comment block, a @copyright and @author tags to mention the file originally came from the author.

An example of such file would be, messenger/index.php. Here is the new comment block on header.
